# 使用：工作区

用户侧只需要记住：**一个项目 = 一个工作区文件夹**。

## 创建

- **CLI**：`litnexus init <目录>` — 写入配置/列表模板/数据目录，并设为活动工作区
- **Mac UI**：项目选择页 / 首次向导中新建或打开

## 日常编辑

| 文件 | 你改什么 |
|------|----------|
| `journals.txt` | 关注期刊 |
| `keywords.txt`（或 `keywords/*.txt`） | 检索式 |
| `litnexus.toml` | AI、分类问题、下载参数等 |

库文件 `litnexus.db` 一般由程序维护；导出在 `exports/`，原始下载在 `downloads/`。

## 备份与迁移

复制整个工作区目录到另一台机器即可；再在客户端「打开」该目录或设为活动工作区。

架构细节见 [工作区与配置](../architecture/workspace.md)。
